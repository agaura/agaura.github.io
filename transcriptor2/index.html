Alphabet

A/a

Default Pronunciation: /æ/; like a as in apple

Other Pronunciations:

  • Before N or M, A is pronounced /eə/; like a as in can
  • Before another vowel or at the end of words, as well as before Ŋ, A is pronounced /eɪ/; like a as in date
    • When paired with I, AI is also pronounced /eɪ/;
  • Before R, A is pronounced /ɑ/; like a as in car

Å/å

Default Pronunciation: /ɑ/; like a as in all

B/b

Default Pronunciation: /b/; like b as in bag

C/c

Default Pronunciation: /kʰ/; like c as in cat

Other pronunciations:

  • When following S, starting unstressed syllables, before consonants, or at the end of a word, C is pronounced /k/; like c as in scar

D/d

Default Pronunciation: /d/; like d as in dog

Other pronunciations:

  • When preceding J or R, D is pronounced /dʒ/; like d as in drag or badge
  • When surrounded by vowels and preceding an unstressed syllable, or when following a vowel at the end of a word, D is pronounced /ɾ/; like d as in bad

Ð/ð

Default Pronunciation: /ð/; like th as in the

E/e

Default Pronunciation: /ɛ/; like e as in egg

Other pronunciations:

  • Before another vowel or at the end of words, E is pronounced /i/; like e as in seed
    • When paired with I, EI is also pronounced /i/;
  • Before R, E is pronounced /e/; like e as in merry

F/f

Default Pronunciation: /f/; like f as in free

G/g

Default Pronunciation: /g/; like g as in girl

H/h

Default Pronunciation: /h/; like h as in hello

Other pronunciations:

  • May be silent at the end of words in order to prevent vowels from changing sound

I/i

Default Pronunciation: /ɪ/; like i as in ill

Other pronunciations:

  • Before another vowel or at the end of words, I is pronounced /aɪ/; like i as in hi
    • When paired with I, II is also pronounced /aɪ/;
  • When paired with A, AI is pronounced /eɪ/; like a as in date
  • When paired with E, EI is pronounced /i/; like e as in seed
  • When paired with O, OI is pronounced /ɔɪ/; like oi as in coin

J/j

Default Pronunciation: /j/; like j as in fjord

Other pronunciations:

  • When paired with D, DJ is pronounced /dʒ/; like j as in judge
  • When paired with S, SJ is pronounced /ʃ/; like sh as in ship
  • When paired with T, TJ is pronounced /tʃ/; like ch as in change
  • When paired with Z, ZJ is pronounced /ʒ/; like s as in pleasure

L/l

Default Pronunciation: /l/; like l as in lake

Other pronunciations:

  • When following a vowel and not preceding another vowel, L is pronounced /ɫ/; like l as in full

M/m

Default Pronunciation: /m/; like m as in match

N/n

Default Pronunciation: /n/; like n as in no

Other pronunciations:

  • When preceding B or P, N is pronounced /m/; like n as in rainbow
  • When preceding C or G, N is pronounced /ŋ/; like n as in sink

Ŋ/ŋ

Default Pronunciation: /ŋ/; like ng as in sing

O/o

Default Pronunciation: /oʊ/; like o as in note

Other pronunciations:

  • When preceding R, O is pronounced /ɔ/; like o as in sore
  • When paired with I, OI is pronounced /ɔɪ/; like oi as in coin
  • When paired with U, OU is pronounced /aʊ/; like ou as in out

Ø/ø

Default Pronunciation: /ʊ/; like ou as in could

Other pronunciations:

  • In unstressed syllables, ø is pronounced /ə/; like o as in shallot

P/p

Default Pronunciation: /pʰ/; like p as in pray

Other pronunciations:

  • When following S, starting unstressed syllables, before consonants, or at the end of a word, P is pronounced /p/; like p as in spar

R/r

Default Pronunciation: /ɹ/; like r as in roger

S/s

Default Pronunciation: /s/; like s as in sir

Other pronunciations:

  • When preceding J or R, S is pronounced /ʃ/; like sh as in ship

T/t

Default Pronunciation: /tʰ/; like t as in top

Other pronunciations:

  • After vowels and before unstressed syllables, T is pronounced /ɾ/; like t as in humanity
  • When isolated at the end of words and before syllabic N, T is pronounced /ʔ/; like t as in kitten
  • When following S, starting unstressed syllables, before S and W, or at the end of a word, T is pronounced /t/; like t as in star
  • When preceding J or R, T is pronounced /tʃ/; like t as in true or batch

U/u

Default Pronunciation: /ʌ/; like u as in strut

Other pronunciations:

  • Before another vowel or at the end of words, U is pronounced /u/; like u as in dude
    • When preceding U, UU is also pronounced /u/;
  • When following O, OU is pronounced /aʊ/; like ou as in out

V/v

Default Pronunciation: /v/; like v as in valve

W/w

Default Pronunciation: /ks/; like x as in axe

X/x

Default Pronunciation: /ks/; like x as in axe

Y/y

Default Pronunciation: /j/; like y as in yes

Z/z

Default Pronunciation: /z/; like z as in zip
